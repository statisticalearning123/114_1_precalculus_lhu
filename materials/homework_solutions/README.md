## 🎯 作業設計原則

每次作業都包含：
- **3 道單選題**：測試基本概念理解
- **5 道是非題**：釐清常見迷思
- **1 道簡答題**：訓練計算能力與完整解題步驟

每次作業配分：**8%**

## 📝 詳解內容

每份詳解包含：
1. **作業資訊**：題型、截止時間、配分
2. **題目與詳解**：
   - 每題完整的解題步驟
   - 重點提醒
   - 常見錯誤
3. **學習重點總整理**：本次作業涵蓋的核心概念
4. **學習建議**：如何避免常見錯誤
5. **延伸學習**：相關課程內容連結
