## Ch2 函數

### 2.1 函數的定義

**函數**: `f(x) = y`

> ⚠️ **重點**：這裡的 `=` 是**命名**的意思，不是一般的等於！

- **`f of x`** (唸作 "f of x")：這是在**命名**這個函數叫做 `f`
- 意思：將 `x` **代入**函數 `f` 之後對應到 `y`

### 2.2 函數代入的例子

#### 例 1：基本代入

若 `f(x) = x²`，則：

- `f(0) = 0² = 0`
- `f(1) = 1² = 1`
- `f(2) = 2² = 4`

#### 例 2：多項式函數

**函數式**: `f(□) = (□)² + (□) + 1`

- `f(0) = 0² + 0 + 1 = 1`
- `f(1) = 1² + 1 + 1 = 3`

或寫成：`f(x) = x² + x + 1`

- `f(0) = 0² + 0 + 1 = 1`
- `f(1) = 1² + 1 + 1 = 3`

#### 例 3：分式函數

若 `f(x) = 1/x`，則：

- `f(1) = 1/1 = 1`
- `f(2) = 1/2`
- `f(0)` ❌ 不行（分母不能為 0）

---

### 2.3 函數可以有不同的命名方式

#### 方法 1：用不同字母命名

```
f(x) = x²
g(x) = 1/x
h(x) = √x
```

`f`, `g`, `h` 是**不同的函數**，用不同字母來區分。

#### 方法 2：用下標編號命名

如果有多個函數想用同一個字母，可以用下標：

① `f₁(x) = x²` （第 1 個 f）
② `f₂(x) = 1/x` （第 2 個 f）
③ `f₃(x) = √x` （第 3 個 f）

> 💡 **說明**：f₁, f₂, f₃ 是三個**不同的函數**，只是都用 f 命名，用下標 1, 2, 3 來區分

## 🎯 函數如何代入？？

若 `f(x) = x²` ⇝ `f(2) = ?`

> 💡 **符號說明**：⇝ = 波浪箭頭（想知道）

### 步驟

**Step 1**: 先把未知數 `(x)` 寫成框框或括號

```
f(□) = (□)²
```

**Step 2**: 用要代入的東西取代 □

```
f(2) = (2)²
```

**Step 3**: 計算

```
= 4
```

---

### 練習題

#### 例 1) 若 `f(x) = x²`，則 `f(y) = ?`

**解**:
```
f(y) = (y)² = y²
```

> ⚠️ 注意：不是 `x²`

#### 例 2) 若 `f(x) = x²`，則 `f(x²) = ?`

**解**:
```
f(x²) = (x²)² = x⁴
```

> ⚠️ 注意：`(x²)²` $\rightarrow$ `x⁴`

## 2.4 指數律 （指數的規則）

### 定義

**n次方**: 乘自己 n 次

```
aⁿ = a·a·a...a  (n次方：底數乘 n次)
```

### 例子

- `x⁴ = x·x·x·x` (4個 x)
- `x³ = x·x·x` (3個 x)
- `5³ = 5×5×5`

---

### 🔑 指數律 (重要)

#### 規則 1: 同底數相乘

```
xᵃ · xᵇ = xᵃ⁺ᵇ
```

**例**:
```
x³ × x² = (x·x·x)·(x·x) = x⁵
```

---

#### 規則 2: 指數的指數

```
(xᵃ)ᵇ = xᵃ·ᵇ
```

**證明**:

例 1:
```
(x²)² = x² · x² = (x·x)·(x·x) = x⁴
```

例 2:
```
(x²·x)³ = x³ · x³ = (x·x·x)·(x·x·x) = x⁶
```

例 3:
```
(x²·x)² = x² · x² · x² = x⁶
```

---

### 📦 總結

```
框框 1: (xᵃ)ᵇ = xᵃ·ᵇ
框框 2: xᵃ · xᵇ = xᵃ⁺ᵇ
```

---

### 進階練習

#### 計算

① `(2³)³ = 2³·³ = 2⁹ = 512`

   或 `2³ × 2³ × 2³ = 8 × 8 × 8 = 512`

② `2² × 2³ = 2²⁺³ = 2⁵ = 32`

③ 進階:

   ```
   ((2³)²)² = (4²)² = (16)² = 256
          = 2^(3×2×2) = 2⁸ = 256 $\checkmark$
   ```

---

### 函數代入的進階題

#### 題目

若 `f(x) = x³`，則 `f(x²) = ?`

**解**:
```
f(x²) = (x²)³ = x²·³ = x⁶
```

### P.19 例 4

> ⚠️ 注意：`3×x = 3·x`

**題目**: 若 `f(x) = x² - 3x + 1`，則：

(1) `f(4) = ?`

**解**:
```
f(4) = (4)² - 3(4) + 1
     = 16 - 12 + 1
     = 5
```

(2) `f(3) = ?`

**解**:
```
f(3) = (3)² - 3(3) + 1
     = 9 - 9 + 1
     = 1
```

**幹**：`f(4) - f(3) = 5 - 1 = 4` $\checkmark$ **答案**

## 💡 2. 看到不懂的東西就當作一個東西

### 例題

若 `f(x) = x² - 3x + 1`

求：`f(x+Δx) - f(x) = ?`

#### Δ: Delta (唸作 "delta"，表示變化量)

**解**:
```
f(x+Δx) = (x+Δx)² - 3(x+Δx) + 1
```

**展開**:
```
= (x+Δx)² - (3x+3Δx) + 1
= (x+Δx)² - 3x - 3Δx + 1
```

---

### 🚨 負號規則（重要）

#### 減法 vs 負數

**減 (v.)**: `0-2` (複式2) $\rightarrow$ 0和2相減，非遞，是算式
**負 (adj)**: `負2` $\rightarrow$ `-2` 是一個數

#### 規則

```
減 -(a+b) = -a - b
   唸：減括號 a 加 b 等於 負 a 減 b

負 (-1) × (a+b) = (-1)·a + (-1)·b = -a - b
   唸：負一 乘 括號 a 加 b 等於 負一 乘 a 加 負一 乘 b 等於 負 a 減 b
```

**例**:
```
-(-2) - (-2) 是多少？
= +2 + 2 = 4
正 加
```

## 2.5 乘法公式（括號上）

### ❓ Q: `(2+3)² = 2² + 3² + ?`

### ❌ 錯誤想法

「先加再平方 = 先平方再加」❌

### $\checkmark$ 正確答案

```
(2·3)² = 2² · 3² $\checkmark$$\checkmark$$\checkmark$ (這個對了)
```

但 `(2+3)² ≠ 2² + 3²`

---

### 🎨 幾何圖解（搭配優先）

```
      2        3
   ┌────┬─────────┐
   │ 2² │  2×3   │ 2
   ├────┼─────────┤
   │2×3 │  3²    │ 3
   └────┴─────────┘
```

**面積計算**:

```
(2+3)² = 2² + 3² + 2×(2×3)
       = 4 + 9 + 12
       = 25
```

---

### 📐 完全平方公式

```
      a        b
   ┌────┬──────┐
 a │ a² │  ab  │
   ├────┼──────┤
 b │ ab │  b²  │
   └────┴──────┘
```

**公式**:

```
(a+b)² = a² + b² + 2ab
        ─┬─  ─┬─  ──┬──
       短邊  長邊  兩相乘
       A平方 B平方 取相乘
```

## 🧮 完整計算：f(x+Δx) - f(x)

### 完整展開

```
f(x+Δx) = (x+Δx)² - 3(x+Δx) + 1
```

**使用乘法公式**:

```
(a+b)² = a² + 2ab + b²
```

所以：

```
(x+Δx)² = x² + 2x·Δx + (Δx)²
```

**代入**:

```
f(x+Δx) = (x+Δx)² - 3(x+Δx) + 1

        = [x² + 2x·Δx + (Δx)²] - 3(x+Δx) + 1

        = x² + 2x·Δx + (Δx)² - 3x - 3Δx + 1
          ↑    ↑       ↑       ↑    ↑      ↑
          x²  (Δx)²   x·Δx     x    Δx   常數
```

---

### 計算 f(x+Δx) - f(x)

```
f(x+Δx) - f(x) = [x² + 2x·Δx + (Δx)² - 3x - 3Δx + 1]
                - [x² - 3x + 1]
```

**展開負號**：

```
= x² + 2x·Δx + (Δx)² - 3x - 3Δx + 1
  - x² + 3x - 1
```

**合併同類項**（相同項消去）：

```
= (Δx)² + 2x·Δx - 3Δx
```

因此

```
f(x+Δx) - f(x) = (Δx)² + 2x(Δx) - 3(Δx)
```
