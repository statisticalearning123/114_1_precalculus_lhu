## 例題 4（複習）

若函數 $f(x) = x^2 - 3x + 1$，求：

(1) $f(4) - f(3) = ?$

(2) $f(x + \Delta x) - f(x) = ?$

(3) $\dfrac{f(x + \Delta x) - f(x)}{\Delta x} = ?$

---

### 解答

**(1)**

$f(4) - f(3) = (4^2 - 3 \times 4 + 1) - (3^2 - 3 \times 3 + 1) = 5 - 1 = 4$

**(2)**

$f(x + \Delta x) - f(x) = [(x + \Delta x)^2 - 3(x + \Delta x) + 1] - (x^2 - 3x + 1)$

$= x^2 + 2x\Delta x + (\Delta x)^2 - 3x - 3\Delta x + 1 - x^2 + 3x - 1$

$= 2x\Delta x + (\Delta x)^2 - 3\Delta x$ 錯同類項

$= (2x - 3)\Delta x + (\Delta x)^2$

**(3)**

$\dfrac{f(x + \Delta x) - f(x)}{\Delta x} = \dfrac{(2x - 3)\Delta x + (\Delta x)^2}{\Delta x} = (2x - 3) + \Delta x$

## 乘法公式

### 長方形面積圖示

**完全平方**：

$(a+b)^2 = a^2 + 2ab + b^2$

例：$70^2 = 49 + 2 \times 49 + 2 \times 49 = 4900$

**完全平方差**：

$(a-b)^2 = a^2 - 2ab + b^2$

例：$50^2 = 2500$

---

### 平方差公式

$(a+b)(a-b) = a^2 - b^2$

**例**：$79^2 - 21^2 = 5800$

$= (79+21)(79-21)$

$= 100 \times 58$

---

### 練習

① $99^2 - 1^2 = (99+1)(99-1) = 9800$

② $23^2 + 77^2 + 2 \times 23 \times 77 = (23+77)^2 = 100^2 = 10000$

③ $100^2 + 50^2 - 2 \times 100 \times 50 = (100-50)^2 = 50^2 = 2500$

④ $x^2 - y^2 = (x+y)(x-y)$（配方分解）

⑤ $x^4 - y^4 = (x^2+y^2)(x^2-y^2) = (x^2+y^2)(x+y)(x-y)$

⑥ $x^8 - y^8 = (x^4+y^4)(x^4-y^4) = (x^4+y^4)(x^2+y^2)(x+y)(x-y)$

⑦ $(\square + \triangle)^2 = \square^2 + \triangle^2 + 2\square\triangle$（最簡）

⑧ $(x^2 + x^3)^2$（展開）

$= (x^2)^2 + (x^3)^2 + 2x^2 \cdot x^3$

$= x^4 + x^6 + 2x^5$

（一次只做一件事）

## 合併同類項

### Step 1：判斷是哪一類（同類才能合併，不同類不能合併）

$2 + 3 = 5$ 可

$x + 2$ 不可

$x + 3x = 4x$ 可

$x + 3x^2$ 不可

$1x + 0x = (1+0)x$ 可

$x + yx$ 不可當作同類項

$ax^2 + bx + c$

說明：
- $a$：常數（只是不知道）
- $y$：未知數

---

### 情況1：只有一個變數

類別：$1, x, x^2, x^3, x^4, \ldots, x^n$ $\Rightarrow$ $x$ 的整數次方

---

### Step 2：把同類項寫在一起，合併

$(\quad)x^n + (\quad)\cdots + (\quad)x^4 + (\quad)x^3 + (\quad)x^2 + (\quad)x + (\quad)$

---

### Step 3：降冪排列（大到小）

$(\quad)x^n + (\quad)\cdots + (\quad)x^4 + (\quad)x^3 + (\quad)x^2 + (\quad)x + (\quad)$

## 例題

### 例①

$(x^2+2x) + (x^2+2) + 1 + (x^2+4x+4) + (x^2+8x+4)$

$= 4x^2 + (2+2+4+6)x + (1+4+9)$

$= 4x^2 + 14x + 14$

---

### 例②

$(x+1)^2 + (x-1)^2$

$= (x^2+1+2x) + (x^2+1-2x)$

$= 2x^2 + 0x + 2 = 2x^2 + 2$
