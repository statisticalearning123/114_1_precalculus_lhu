# 考試資料夾

本資料夾包含所有考試相關的資料，包括考卷、公式表、詳解等。

## 📁 資料夾結構

```
exams/
├── midterm/          # 期中考相關資料
│   ├── 期中考公式表.md
│   ├── 期中考公式表.pdf
│   ├── Makefile
│   └── README.md
└── final/            # 期末考相關資料（未來新增）
```

## 📚 使用說明

### 期中考公式表

期中考公式表位於 `midterm/` 資料夾，包含 Ch2 函數的所有重要公式。

詳細使用方式請參閱：[midterm/README.md](midterm/README.md)

### 生成 PDF

```bash
cd materials/exams/midterm
make
```
