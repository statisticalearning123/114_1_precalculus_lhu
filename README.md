# 預備微積分課程 - 114學年度第1學期

龍華科技大學預備微積分課程

## 📂 目錄說明

- `_template/` - Quarto 課程網站範本
  - 完整的課程網站結構
  - 包含範例頁面和內容
  - 可作為參考或直接使用
  - 詳細說明請參閱 `_template/README.md`

- `上課筆記/` - 課程筆記和教學資料
  - 存放上課筆記、講義等文件

## 🚀 使用範本

如果要使用範本建立課程網站：

```bash
# 複製範本檔案到主目錄
cp -r _template/* .

# 然後開始客製化內容
```

## 📝 建立自己的課程網站

你可以根據自己的課程規劃，自由組織這個目錄的內容。
